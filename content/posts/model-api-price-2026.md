## 价格战的起点

2024 年 5 月，字节跳动率先将豆包大模型的 API 价格降到每百万 token 0.8 元，引发了一场持续至今的价格战。两年后的今天，国产大模型 API 的价格已经降了 90% 以上。

2026 年 Q2 的价格格局可以用一句话概括：**基础能力趋近免费，差异化能力开始收费。**

## 免费额度：各家的"试用装"

除了降价，各家还提供了大量免费额度：

- **火山引擎**：新用户注册送 500 万 token，部分模型限时免费
- **阿里云百炼**：Qwen 系列每月 100 万 token 免费
- **智谱 AI**：GLM 系列新用户送 500 万 token
- **Kimi API**：新用户送 15 元额度
- **DeepSeek**：开源模型可自部署，API 价格已是国产最低

对于个人开发者和小团队来说，**用免费额度做 MVP 验证，已经是零成本的事。**

## 价格战的终局

价格战不会无限持续。当基础模型能力趋同、价格趋近于零时，竞争会转向：

1. **模型差异化**：代码、Agent、多模态、长上下文——细分能力成为定价依据
2. **工具链整合**：模型 + 向量数据库 + 工具调用 + 部署——打包服务比单卖 API 更有价值
3. **行业解决方案**：通用模型价格战，垂直领域模型溢价

对开发者来说，选型建议是：**基础任务用最便宜的，关键任务用最合适的，不要只看价格。**

> 数据截至 2026 年 7 月，价格可能随时变动，请以各平台官方公告为准。
