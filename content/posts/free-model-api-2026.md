## 免费额度的时代

2024 年之前，调用大模型 API 是一件昂贵的事。GPT-4 的价格让很多个人开发者望而却步。

2026 年的今天，情况完全变了。国产大模型平台为了争夺开发者，纷纷推出大量免费额度。**对个人开发者来说，调用大模型 API 的成本已经降到了接近零。**

## 各平台免费额度一览

### 火山引擎（豆包/Seed）

- **注册赠送**：500 万 token
- **限时免费**：部分模型长期免费调用
- **Agent Plan**：49 元/月，包含多模型调用额度
- **特点**：字节系生态，扣子 Coze + TRAE 联动

### 阿里云百炼（Qwen）

- **注册赠送**：Qwen 系列 100 万 token/月
- **开源模型**：Qwen3 开源版本可自部署，零成本
- **特点**：工具调用能力强，Agent 场景表现优秀

### 智谱 AI（GLM）

- **注册赠送**：500 万 token
- **学生优惠**：认证学生额外赠送额度
- **特点**：中文理解能力突出，学术场景友好

### Kimi API（月之暗面）

- **注册赠送**：15 元额度
- **长上下文**：128K 上下文窗口
- **特点**：长文档处理能力强

### DeepSeek

- **API 价格极低**：输入 0.5 元/百万 token，已是国产最低
- **开源可部署**：DeepSeek V4 开源，自部署零成本
- **特点**：推理能力强，性价比之王

### MiniMax

- **注册赠送**：新用户送额度
- **M3 模型**：100 万 token 上下文，Agent 能力突出
- **特点**：多模态支持好

## 开源模型自部署

如果你有自己的服务器（哪怕是 2 核 4G 的轻量云），可以自部署开源模型，完全零成本：

| 模型 | 最低显存 | 适合场景 |
|------|---------|---------|
| Qwen3-1.8B | 4GB | 轻量对话、文本处理 |
| DeepSeek-V4-Lite | 8GB | 通用对话、代码生成 |
| GLM-5-9B | 16GB | 长上下文、复杂推理 |

用 Ollama 或 vLLM 部署，10 分钟内就能跑起来。
