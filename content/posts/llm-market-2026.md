## 一句话结论

2026 年 7 月，国产大模型 API 价格已进入**「厘时代」**——DeepSeek-V3 低至 0.001 元/千 tokens，字节豆包降幅 99.6%。**基础模型价格已触底，差异化竞争转向 Agent 能力与多模态**。

## 反常方向：涨价阵营

并非所有厂商都在降价。2026 年上半年出现**分化**：

- **腾讯云**：连续两次上调 API 价格（3 月和近期），部分服务涨价约 30%
- **智谱**：多次上调价格
- **阿里 DataWorks**：取消 API 调用量限制，但免费额度缩减（标准版 10 万次/月，专业版 50 万次/月）

> **原因**：基础模型价格已触底，涨价反映的是"从铺量到盈利"的转折。单 Token 价格上升，但单 Token 创造的价值在提升。

## 开源逆袭

- **DeepSeek-V3**：开源模型首次在中文场景反超闭源
- **Llama 4**：Meta 开源旗舰，多模态原生
- **Qwen3**：阿里开源，中文社区第一选择
- **MiMo-V2.5**：小米开源，缓存命中 $0.0028/M tokens

开源模型 + 自部署已成为**企业级 AI 落地的新默认选项**。

## 2026 下半年趋势展望

1. **基础模型价格触底**，未来降幅有限
2. **差异化转向**：Agent 工具调用、RAG 长期记忆、多模态推理
3. **推理成本**仍是核心竞争点——7 月 1 日三家同时宣布降半价/砍 60%
4. **企业级落地加速**：推理降价 + 派驻工程师 + 行业模板
5. **合规与数据安全**成为企业选型关键因素

## 给开发者的建议

1. **不要绑定单一模型**：同时接入 2-3 个（如 DeepSeek + 通义 + 豆包），做 A/B 切换
2. **利用缓存降低重复成本**：上下文缓存在百炼等平台可显著减少开销
3. **小模型做 80% 的事**：DeepSeek-Flash / 豆包 Turbo 等轻量版，价格低一半
4. **关注开源自部署**：长期看比 API 调用更省钱，且数据不出域
5. **审批预算趁现在**：当前是 2023 年以来价格最低点
