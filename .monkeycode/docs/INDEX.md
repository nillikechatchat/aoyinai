# 敖胤AI 项目文档

本目录记录敖胤AI静态博客的架构、页面与数据接口，以及常用开发流程。内容面向维护站点文章、赛事数据和页面组件的开发者。

## 核心文档

| 文档 | 内容 |
| --- | --- |
| `ARCHITECTURE.md` | 技术栈、子系统、数据流和构建流程 |
| `INTERFACES.md` | 页面路由、内容函数、赛事数据模型与外部服务 |
| `DEVELOPER_GUIDE.md` | 环境要求、构建命令与内容维护规则 |

## 阅读顺序

1. 阅读 `ARCHITECTURE.md` 了解静态站点的构建与数据流。
2. 阅读 `INTERFACES.md` 定位页面、文章和赛事模块。
3. 阅读 `DEVELOPER_GUIDE.md` 执行内容维护、构建验证和提交检查。
